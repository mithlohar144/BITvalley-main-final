# BitVally - Website & Software Development Company

## Overview
BitVally is a modern website for a software development company built with HTML, CSS, JavaScript and GSAP animations. The website features a clean design, smooth animations, and responsive layout.

## Features
- Responsive design that works across desktop, tablet and mobile devices
- Custom cursor animation using GSAP
- Dark/Light theme toggle
- Smooth scrolling using Locomotive Scroll
- Testimonial carousel
- Service cards with hover effects
- Portfolio showcase
- Contact form
- Newsletter subscription
- Modern wave animations

## Technologies Used
- HTML5
- CSS3 (with Tailwind CSS)
- JavaScript 
- GSAP (GreenSock Animation Platform)
- Locomotive Scroll
- Remix Icons
- Google Fonts (Inter, Archivo)

## Setup & Installation
1. Clone the repository:
```bash
git clone https://github.com/yourusername/BITvalley.git
```

2. Open the project folder:
```bash 
cd BITvalley
```

3. Open index.html in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```
